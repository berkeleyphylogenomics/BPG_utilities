Welcome to the PhyloFacts FAT-CAT Webserver Quick-Start Guide

 

 

The FAT-CAT webserver uses hidden Markov models (HMMs), to place sequences into PhyloFacts trees for functional annotation and ortholog identification. This guide will get you started using FAT-CAT. It will show you how to submit a query sequence and interpret the results.

 

The FAT-CAT home page is simple to use in the default settings. However, there are tunable parameters for expert users

Figure 1 The FAT-CAT Webserver home page with important features for query submission highlighted with red letters

 

A  Sequence header box: The query sequence should be in unaligned FASTA format. Paste the querys header in this box.

 

B  Raw sequence: Paste the query sequence in this box. FAT-CAT only accepts protein sequences that have no special characters (B,J,O,U,X,Z). The maximum length of the sequence should be 2000 residues. FAT-CAT does not accept DNA sequences.

 

 

C Email: Providing an email address is optional. This way you will get an email with a link to your final results. Or you can bookmark the submitted page and return to view the results later.

 

D Email subject: This will allow you to specify a meaningful subject line for the email with a link to your results

 

E Comments: This will allow you to add comments about you FAT-CAT job that will be recorded in your job summary

 

 


F Submit: Click on the                   button to submit your sequence and wait for the results.

Congratulations!! You have successfully used the FAT-CAT webserver
