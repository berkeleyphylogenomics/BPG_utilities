Searching UniProt for an allowed ID/accession.

For our example (GenBank accession DAA01234, which is not an allowed identifier for a PHOG search), we mention a couple options for how to proceed.

One option is to search UniProt for an allowed ID/accession (http://www.uniprot.org).

Just enter the identifier in the UniProt search box and click the search button.

This example yields identifiers that are allowed for a PHOG query. Either the UniProt Accession or ID ("Entry name" on the UniProt web page shown here) can be used to initiate a PHOG search as previously described.
