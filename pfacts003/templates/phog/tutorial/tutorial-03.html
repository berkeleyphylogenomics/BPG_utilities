The PhyloFacts Orthology Group Web Server takes as input a query protein sequence and delivers predicted orthologs.

The PHOG Server takes as input your query protein sequence as a sequence identifier or in FASTA sequence format.

The output depends on the form of input (ID/accession or FASTA).

In the case where a sequence ID or accession is provided as input, we search for all orthology groups containing the sequence. If any are found, we produce a report with data about these predicted orthologs (jump to page 8).

If a sequence is given in FASTA format, we run BLAST to search for any homologs to that sequence that are placed in any orthology groups. Users can then select a homolog to view its pre-computed orthologs (jump to page 19).

We describe both forms of output in more detail later.
