Table of BLAST matches for a query sequence in FASTA format. Note: these are not predicted orthologs.

For queries based on FASTA sequence input, a table of BLAST matches is displayed, ordered by BLAST score.

Note that these are not predicted orthologs, but rather a list of matches that can be used to initiate a PHOG search. Simply click on the "orthologs" link for a match of your choice to initiate a search for PHOGS containing that match.

For each match, the following information is listed in the table.

  • The gene ID/accession and a link to the associated web page.
  • A link to initiate a PHOG query based on the gene ID/accession (with results as described earlier).
  • The scientific and common species names.
  • A description of the protein.
  • The percent identity to the query sequence.
  • The BLAST E-value.
  • The BLAST bit score.
