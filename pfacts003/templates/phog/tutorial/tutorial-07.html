Table of PHOGs containing a query sequence specified by ID/accession.

For each PHOG that contains the query sequence (when input by ID/accession), the following information is provided in a table.

  • The PHOG identifier and a link to the associated PHOG web page.
  • A brief description of the PHOG.
  • Gene Ontology evidence from UniProt, with links to the Gene Ontology term details and evidence code descriptions (the best evidence code is shown in parentheses).
  • Pfam domains, with links to the associated Pfam web pages.
  • The taxonomic distribution, with links to the UniProt taxonomic descriptions.
  • The total number of seqences in the PHOG.
