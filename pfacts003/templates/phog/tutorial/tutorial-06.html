Selecting the evolutionary distance and initiating the ortholog search.

[1] You can select the evolutionary distance threshold value by either selecting a pre-loaded value from the drop-down box or by entering a custom value in the threshold entry box. ([2] Click the "evolutionary distance" link to see a description of the pre-loaded values.) We give an example of results dependant on the evolutionary distance parameter later (jump to page 9).

[3] To initiate the ortholog search, click the "Search" button adjacent to the ID/accession entry box. [4] When you click the "Search" button, you will notice a graphic indicating the search is ongoing. Once the search is complete, the page will refresh with the results listed below the data entry area.

Results based on input IDs/accessions are displayed in two tables, one describing the PHOGs which contain the query sequence and the other describing the sequences predicted to be orthologs or in-species paralogs.
