Table of orthologs and in-species paralogs for a query sequence specified by ID/accession.

The next table has a row for each ortholog and in-species paralog from one or more PhyloFacts families, with the following information.

  • The gene ID/accession and a link to the associated web page.
  • The scientific and common species names.
  • A description of the protein.
  • A link to the SwissProt curated sequence, if available.
  • A link to Gene Ontology annotations with experimental evidence, if available.
  • The IntEnz (EC) classification at the EBI IntEnz database and link to the associated EC page, if available.
  • A link to the associated Kyoto Encyclopedia of Genes and Genomes (KEGG) web page, if available.
  • A link to the associated BioCyc metabolic pathway page, if available.
  • A link to the PhyloFacts interacting proteins, if available.
  • The Pfam domains found in the family, linked to the associated Pfam web pages.
  • A link to the PhyloFacts family containing the sequence.
  • The alignment type, whether global or local.
  • The PHOG containing the sequence, linked to the associated PHOG web page.
  • Links to view the PhyloFacts tree for either the full family or the PHOG. (For help with the tree viewer click the "Need help?" link at the top of the viewer page, or click here.)
  • Links to UniProt literature associated with the sequence, if available.

Note that the table may consist of multiple pages which can be navigated via the links at the bottom of the table. The table can also be downloaded in CSV format (comma separated values) by clicking the "Download as CSV" button below the table.
